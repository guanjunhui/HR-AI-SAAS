# 前端功能完善实施计划

根据后端已实现的批量导入和任职统计功能，完善前端对应的交互功能。

## 1. 类型定义扩展 (hr.ts)
新增以下类型：
- `EmploymentEventStatisticsDTO` - 统计响应类型（按类型统计、按状态统计）
- `EmploymentEventStatisticsStatItem` - 统计项类型

## 2. API 服务扩展 (hr.ts)
新增以下 API 函数：
- `importEmploymentEventsApi(file)` - POST 上传 Excel 批量导入
- `downloadEmploymentEventTemplateApi()` - GET 下载导入模版
- `getEmploymentEventsStatisticsApi()` - GET 获取统计数据

## 3. 页面功能增强 (events/index.tsx)
### 3.1 导入功能
- 新增"导入"按钮，触发上传弹窗
- 导入弹窗：支持拖拽或点击上传 Excel 文件
- 上传后显示处理结果（成功/失败数量）
- 新增"下载模版"按钮

### 3.2 统计功能
- 在页面顶部或独立区域展示统计卡片
- 统计维度：
  - 按事件类型：入职、转正、调岗、晋升、降级、调薪、离职
  - 按状态：草稿、待审批、已通过、已驳回、已取消
- 点击统计卡片可跳转到对应筛选结果

## 4. UI 组件依赖
- 使用 `Upload` 组件处理文件上传
- 使用 `Card` `Row` `Col` 展示统计卡片
- 复用现有组件风格

- [ ] 导入按钮可点击，显示模态框
- [ ] 支持 Excel 文件上传，调用后端 API
- [ ] 下载模版功能正常
- [ ] 统计卡片展示按类型和按状态的统计数据
